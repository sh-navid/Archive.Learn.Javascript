# JQuery
## Selectors
- `*`
- Attribute
    - tag[attr|="NAME"]
    - tag[attr*="NAME"]
    - tag[attr^="NAME"]
        - _Begin with word `attr`_
    - tag[attr$="NAME"]
        - _Ends with word `attr`_
    - tag[attr~="NAME"]
        - _Contains word `attr`_
    - tag[attr!="NAME"]
        - _Do Not have `attr` or `value`_
