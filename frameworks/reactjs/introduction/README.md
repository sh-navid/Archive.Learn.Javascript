# ReactJS
## Introduction
- What is React?
    - A JS Library
    - To make reusable UI Components
    - Uses JSX
        - `JavaScript XML`
- Setup
    - `npx create-react-app my-app`
    - You may need to export path of new node version in ubuntu
        - `export PATH=/home/ns/<node folder>/bin/:$PATH`
- Run
    - `cd my-app`
    - `npm start`
