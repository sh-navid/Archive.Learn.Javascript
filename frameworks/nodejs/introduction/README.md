# NodeJS
## Introduction
- Installation
    - Windows
        - Download LTS version of NodeJs from [here](https://nodejs.org/en/).
    - Ubuntu
        - Open a terminal and Enter
            - `apt install nodejs`
- Setup project
    - First Step
        - `npm init`
            > You mau want to use `npm init -y`  
            > To skip questions and name your project as current working folder
    - Second Step
        - `npm install`
